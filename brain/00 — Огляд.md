# 00 — Огляд

## Що це

Telegram-бот для абітурієнтів України, який бере конкурсні списки з vstup.osvita.ua і показує користувачу:

- скільки людей реально йде попереду нього на бюджет (а не «всього 2278 заяв, ти 14-й»)
- який його шанс на цю спеціальність (🟢 / 🟡 / 🔴 / ⚫)
- історію кожного абітурієнта через abit-poisk.org.ua (куди ще подавався, де вже пройшов)

Це продовжує ідею Python-бота [AbitAssistant_Bot](https://github.com/OlexiyOdarchuk/AbitAssistant_Bot) (v2.x), але переписане з нуля на Go з модульною архітектурою і Telegram-первинним UX.

## Чому переписували

Python-моноліт виріс. На червень 2026 хотіли:

1. **Модульну архітектуру** — чисте доменне ядро `internal/abit` (presentation-agnostic), щоб поверх нього легко жили різні фронти: бот, CLI, а далі веб. У Python усе жило в `app/` моноліті. (Спочатку ядро лежало в `pkg/` як `go get`-абельна бібліотека, але зовнішнього споживача немає — дропнули `pkg/`, усе в `internal/`. → [[01 — Архітектура]].)
2. **Менший runtime** — scratch Docker ~13 МБ замість `python:slim` ~400 МБ.
3. **Конкурентність на горутинах** — fan-out scraping osvita через 8 workers, замість asyncio.
4. **Type safety end-to-end** — sqlc-згенеровані запити, typed callback-args, не `dict-of-anything`.

Деталі рішень → [[01 — Архітектура]].

## Мотивація (особиста)

Автор сам здає НМТ 2025, проходить через вступ. Бот спочатку був ad-hoc Python скриптом для свого профілю, потім [@AbitAssistant_bot](https://t.me/AbitAssistant_bot). У 2026 — це open source, GPLv3, з ботом для всіх абітурієнтів, які знають про нього.

## Хто користується

- Абітурієнти (через Telegram-бот; далі — веб)
- Я сам (як автор + admin розсилки)

## Що НЕ робимо

- Не парсимо ЄДЕБО напряму — поки немає публічного API, реверс зашифрованих ПІБ потребує живої сторінки. Деталі → [[06 — EDBO Research]].
- Не зберігаємо PII довше ніж треба — saved-lists мають opaque share-token, кеші мають TTL з auto-vacuum.
- Не робимо matplotlib-стиль гістограм — pure-Go `go-chart/v2` без CGo, бо scratch Docker.

## Зв'язок з Python v2

Python-репо тепер архівований — `/home/ishawyha/Documents/Coding/Python/AbitAssistant_Bot/`. Його README перероблено в deprecation notice з лінком сюди. Telegram-бот `@AbitAssistant_bot` той самий, переключений на Go.
